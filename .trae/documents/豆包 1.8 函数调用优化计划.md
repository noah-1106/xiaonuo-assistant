# 豆包1.8函数调用优化计划

## 问题分析

根据官方文档和当前代码实现，系统存在以下问题：

1. **消息格式不完整**：构建的 `aiResponseMessages` 缺少 assistant 消息中的 `tool_calls` 字段
2. **消息历史不完整**：没有包含之前的对话历史，影响多轮对话上下文
3. **工具调用结果格式**：工具调用结果的格式可能不符合官方要求
4. **缺少Strict模式**：未启用严格模式，可能导致参数格式不一致
5. **错误处理不完善**：工具调用失败时的错误处理机制需要优化
6. **函数定义不规范**：函数定义可能不符合官方推荐的格式

## 优化方案

### 1. 修正消息构建逻辑
- **修改 chatController.js**：在构建 `aiResponseMessages` 时，确保 assistant 消息包含完整的 `tool_calls` 字段
- **保持消息顺序**：严格遵循 assistant（含tool_calls）→ tool（含tool_call_id和结果）→ assistant（总结）的顺序

### 2. 构建完整消息历史
- **获取对话历史**：在调用AI之前，获取完整的对话历史记录
- **格式化历史消息**：将历史消息转换为符合官方格式的消息列表
- **保持上下文连续性**：确保多轮对话中的上下文信息不丢失

### 3. 优化工具调用结果格式
- **标准化工具结果**：确保工具调用结果以自然语言形式返回
- **正确设置tool_call_id**：确保每个工具调用结果都正确关联对应的 tool_call_id
- **错误处理优化**：将工具调用失败的错误信息以自然语言形式返回给模型

### 4. 启用Strict模式
- **修改函数定义**：在函数定义中添加 `strict: true` 参数
- **优化参数Schema**：确保参数定义符合JSON Schema标准，添加 `additionalProperties: false`
- **明确必填参数**：在 `required` 数组中明确列出所有必填参数

### 5. 完善错误处理机制
- **工具调用错误**：捕获工具调用过程中的错误，并将错误信息作为工具结果返回
- **参数解析错误**：处理参数解析失败的情况，返回友好的错误信息
- **模型调用错误**：处理模型调用失败的情况，确保系统稳定性

### 6. 优化函数定义规范
- **函数命名**：使用清晰、一致的函数命名规范
- **参数描述**：完善参数的描述信息，避免歧义
- **函数边界**：明确不同函数的边界情况，避免函数调用冲突

## 技术实现要点

1. **消息格式严格遵循官方规范**：
   ```javascript
   // 正确的消息格式
   {
     "role": "assistant",
     "content": "正在处理您的请求...",
     "tool_calls": [
       {
         "id": "tool_call_id",
         "type": "function",
         "function": {
           "name": "function_name",
           "arguments": "{\"param1\": \"value1\"}"
         }
       }
     ]
   }
   ```

2. **工具结果格式**：
   ```javascript
   {
     "role": "tool",
     "tool_call_id": "tool_call_id",
     "content": "工具执行结果或错误信息"
   }
   ```

3. **Strict模式配置**：
   ```javascript
   {
     "type": "function",
     "function": {
       "name": "function_name",
       "description": "函数描述",
       "strict": true,
       "parameters": {
         "type": "object",
         "properties": {
           "param1": {
             "type": "string",
             "description": "参数描述"
           }
         },
         "required": ["param1"],
         "additionalProperties": false
       }
     }
   }
   ```
