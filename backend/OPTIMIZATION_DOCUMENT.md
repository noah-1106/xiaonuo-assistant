## 实现细节

### 函数调用流程

1. **用户发送请求**：用户发送自然语言请求，系统接收并处理
2. **AI分析请求**：AI分析用户请求，判断是否需要调用工具
3. **工具调用**：AI返回tool_calls，系统执行相应的工具函数
4. **工具执行**：系统执行工具函数，获取执行结果
5. **结果回填**：将工具执行结果以role=tool的消息形式回填给模型
6. **AI生成回复**：模型根据工具执行结果生成最终回复
7. **返回用户**：将AI的最终回复返回给用户

### 消息格式

**Assistant消息（带工具调用）：**
```javascript
{
  role: 'assistant',
  content: '',
  tool_calls: [
    {
      id: 'tool_call_id',
      type: 'function',
      function: {
        name: 'function_name',
        arguments: JSON.stringify(function_args)
      }
    }
  ]
}
```

**Tool消息（带结果）：**
```javascript
{
  role: 'tool',
  content: '工具执行结果',
  tool_call_id: 'tool_call_id'
}
```

### 错误处理

当工具调用失败时，系统会：
1. 捕获错误并构建错误消息
2. 将错误消息保存到数据库
3. 以role=tool的消息形式回填给模型
4. 模型根据错误信息生成友好的错误回复
